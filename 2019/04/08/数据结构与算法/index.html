数据结构与算法

发表于 2019-04-08 | 更新于 2019-04-16 | 分类于 计算机基础课程

数据结构基本概念

我们如何把现实生活中大量而复杂的问题以特定的数据类型和特定的存储结构保存在主存储器(内存)中,以及在此基础上为实现某个功能(比如查找某个元素，删除某个元素，对所有元素进行排序)而执行的相应操作,这个相应的操作也叫算法
数据结构 = 个体 + 个体的关系
算法 = 对存储数据的操作

算法 定义:解题的方法和步骤

衡量算法的标准
1.时间复杂度
    大概程序要执行的次数，而非执行的时间
2.空间复杂度
    算法执行过程中大概所占用的最大内存
3.难易程度
4.健壮性
程序 = 数据的存储 + 数据的操作 + 可被计算机执行的语言

预备知识

1.指针

  • 定义: 内存单元的编号，从0开始的非负整数
    指针就是地址，地址就是指针
指针变量是存放内存单元地址的变量
指针的本质是一个操作受限的非负整数

  • 分类:

    1.基本类型的指针
2.指针和数组的关系
    • 如何通过被调函数修改主调函数中的变量，将主调函数中的变量的地址作为参数传入被调函数中，并进行修改，被调函数中需 要声明一个指针变量

2.结构体

  • 为什么出现结构体
    为了表示一些复杂的数据，而普通的基本类型变量无法满足要求时
  • 什么叫结构体
    结构体是用户根据自己实际需求自己定义的复合数据类型，里面存放的称之为成员(类里面的称之为属性)
  • 如何使用结构体
    两种方式：
    struct student st = {1000, "aaa", 20}
    struct student *pst = &st;
    1.st.sid
    2.pst->sid
    pst指向的结构体变量中的sid这个成员
  • 结构体的注意事项
    结构体变量不能加减乘除，但可以相互赋值
    普通结构体变量和结构体指针变量作为函数传参的问题

3.动态内存的分配和释放

  • int *pArr = (int *)malloc(sizeof(int) * len) len = 5 一共20个字节.此处对应动态变量，与之对应的静态变量：数据类型+变量名 = 静态变量
  • malloc函数返回的是第一个字节的地址，(同数组处)用第一个字节的地址表示20个字节的变量,而这第一个字节无实际意义(干地址),需要强制将其表示变得有意义
  • pArr可作为第一个元素的数组名，因其是第一个4个字节的地址，此处因int刚好为4个字节
  • 此分配内存的原理是:请求操作系统分配20个字节的内存给程序以操作，即程序获得20个字节内存的控制权
  • malloc:用多少给多少，还可以释放
  • *pArr = a[0] = 4; pArr[1] = a[1] = 10;
  • 释放:free(pArr); 把pArr所代表的动态分配的20个字节的内存释放，即归还控制权给OS
